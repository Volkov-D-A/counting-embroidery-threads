# DMC palette data

`floss_adrianj.csv` is sourced from the MIT-licensed `sharlagelfand/dmc`
project, which documents the data as derived from
`adrianj/CrossStitchCreator`.

`floss_2017.csv` supplements the base file with DMC colours 01-35. These
colours were added to the DMC range after the base CSV used by the app.

`floss_light_effects.csv` supplements the base file with the current DMC Light
Effects `E...` colours. The official DMC source is used for the supported code
set; the HEX/RGB values are display approximations from public conversion
tables. Multicolour effects such as `E130` cannot be represented exactly by a
single HEX swatch.
